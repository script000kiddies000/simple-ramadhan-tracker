# Ramadhan Web App 🌙

Aplikasi Web Harian Interaktif untuk menemani dan memantau Ibadah Anda selama Bulan Suci Ramadhan. Dibuat dengan antarmuka yang modern, responsif, dan elegan berlapis estetika _Glassmorphism_.

## ✨ Fitur Unggulan

1.  **Tracker Mutaba'ah Harian (Checklist & Kalender)**
    *   Berfungsi memantau target Ibadah Harian: Puasa, Sholat Tarawih, Tilawah, Sedekah, dll.
    *   Dilengkapi 2 Mode Tampilan: **"Hari Ini"** (List Checklist) & **"Kalender"** (Grid Kontribusi 30 Hari ala GitHub).
    *   Secara otomatis memunculkan _progress bar_ setiap harinya.
2.  **Jadwal Sholat & Imsakiyah Cerdas (Otomatis GPS & Countdown)**
    *   Mendeteksi letak / kota Anda berada saat ini.
    *   _Live Countdown Timer_ ⏱️ yang menghitung mundur Jam, Menit, & Detik persis menuju masuknya Waktu Sholat selanjutnya.
3.  **Kalkulator Zakat**
    *   Fitur perhitungan _Zakat Fitrah_ (Berdasarkan jumlah keluarga x harga pokok).
    *   Fitur perhitungan _Zakat Mal_ (Berdasarkan penghasilan total tahunan / setara Nishab Emas).
4.  **Kumpulan Doa & Dzikir Harian**
    *   Lengkap dengan lafaz Arab (Font Amiri/Traditional Serif), terjemahan, dan teks latin.
    *   Fitur Pencarian Doa (*Search-able*).
5.  **Tema Terang & Gelap (Light & Dark Mode)**
    *   Aplikasi beradaptasi dengan pencahayaan layar! Klik tombol ujung kiri atas 🌗 untuk menukar warna tema berestetika _Deep Emerald Green_ atau _Soft Mint White_.
6.  **Data Tersimpan Otomatis (Local Storage)**
    *   Setiap checklist puasa, hitungan dzikir/tarawih, pilihan mode terang/gelap, dan bahkan GPS tersimpan secara lokal ke HP/Komputer (*Tutup Browser & Buka lagi tak perlu mengulang*).

## 🛠️ Teknologi yang Digunakan

Proyek Web ini sepenuhnya dibuat dari struktur paling dasar tanpa Framework berat! Murni menggunakan:
*   **HTML5**
*   **CSS3** (Variabel Warna, Flexbox, CSS Grid, Glassmorphism, Responsive Media Queries)
*   **Vanilla JavaScript (ES6)**
*   API Eksternal: 
    *   _Aladhan API_ (Waktu Sholat)
    *   _OpenStreetMap API_ (Deteksi Lokasi & Nama Daerah)
*   _FontAwesome_ (Ikon)
*   _Google Fonts_ (Outfit & Amiri)

## 🚀 Cara Instalasi & Menjalankan

Aplikasi ini bersifat **PWA-ready / Static Website**, artinya Anda hanya butuh sebuah Browser tanpa *installing* Node/NPM.

1.  *Clone* repositori ini:
    ```bash
    git clone https://github.com/USERNAME/ramadhan-web-app.git
    cd ramadhan-web-app
    ```
2.  Buka *file* `index.html` dengan *web browser* Anda (Chrome/Firefox/Safari) atau jalankan ekstensi **Live Server** di VSCode.
3.  Selamat Menyambut Bulan Suci Ramadhan!
